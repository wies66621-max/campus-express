### 2. 用户控制器（UserController.java）

**文件位置**：`src/main/java/com/school/campusexpress/controller/UserController.java`

```java
package com.school.campusexpress.controller;

import com.school.campusexpress.common.Result;
import com.school.campusexpress.dto.LoginDTO;
import com.school.campusexpress.dto.RegisterDTO;
import com.school.campusexpress.entity.User;
import com.school.campusexpress.service.UserService;
import com.school.campusexpress.util.JwtUtil;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

import java.util.HashMap;
import java.util.Map;

@RestController
@RequestMapping("/api/user")
@CrossOrigin
public class UserController {
    
    @Autowired
    private UserService userService;
    
    /**
     * 用户注册
     */
    @PostMapping("/register")
    public Result<?> register(@RequestBody RegisterDTO registerDTO) {
        try {
            userService.register(registerDTO);
            return Result.success("注册成功");
        } catch (Exception e) {
            return Result.error(e.getMessage());
        }
    }
    
    /**
     * 用户登录
     */
    @PostMapping("/login")
    public Result<?> login(@RequestBody LoginDTO loginDTO) {
        try {
            User user = userService.login(loginDTO);
            
            String token = JwtUtil.generateToken(user.getId(), user.getUsername(), user.getRole());
            
            Map<String, Object> data = new HashMap<>();
            data.put("token", token);
            data.put("user", user);
            
            return Result.success(data);
        } catch (Exception e) {
            return Result.error(e.getMessage());
        }
    }
    
    /**
     * 获取用户信息
     */
    @GetMapping("/info")
    public Result<?> getUserInfo(@RequestHeader("Authorization") String token) {
        try {
            Long userId = JwtUtil.getUserIdFromToken(token);
            User user = userService.getById(userId);
            return Result.success(user);
        } catch (Exception e) {
            return Result.error(e.getMessage());
        }
    }
    
    /**
     * 更新用户信息
     */
    @PutMapping("/update")
    public Result<?> updateUser(@RequestBody User user) {
        try {
            userService.updateById(user);
            return Result.success("更新成功");
        } catch (Exception e) {
            return Result.error(e.getMessage());
        }
    }
    
    /**
     * 获取用户列表（管理员）
     */
    @GetMapping("/list")
    public Result<?> getUserList(
        @RequestParam(defaultValue = "1") Integer page,
        @RequestParam(defaultValue = "10") Integer size,
        @RequestParam(required = false) String username
    ) {
        try {
            return Result.success(userService.getUserList(page, size, username));
        } catch (Exception e) {
            return Result.error(e.getMessage());
        }
    }
}
```

**关键接口说明**：
- `POST /api/user/register`：用户注册
- `POST /api/user/login`：用户登录，返回JWT token
- `GET /api/user/info`：获取当前用户信息
- `PUT /api/user/update`：更新用户信息
- `GET /api/user/list`：获取用户列表（管理员权限）

### 3. 用户服务实现（UserServiceImpl.java）

**文件位置**：`src/main/java/com/school/campusexpress/service/impl/UserServiceImpl.java`

```java
package com.school.campusexpress.service.impl;

import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;
import com.baomidou.mybatisplus.extension.plugins.pagination.Page;
import com.baomidou.mybatisplus.extension.service.impl.ServiceImpl;
import com.school.campusexpress.dto.LoginDTO;
import com.school.campusexpress.dto.RegisterDTO;
import com.school.campusexpress.entity.User;
import com.school.campusexpress.mapper.UserMapper;
import com.school.campusexpress.service.UserService;
import com.school.campusexpress.util.PasswordEncoder;
import org.springframework.stereotype.Service;
import org.springframework.util.StringUtils;

@Service
public class UserServiceImpl extends ServiceImpl<UserMapper, User> implements UserService {
    
    @Override
    public void register(RegisterDTO registerDTO) {
        // 检查用户名是否已存在
        LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(User::getUsername, registerDTO.getUsername());
        User existUser = getOne(wrapper);
        
        if (existUser != null) {
            throw new RuntimeException("用户名已存在");
        }
        
        // 创建新用户
        User user = new User();
        user.setUsername(registerDTO.getUsername());
        user.setPassword(PasswordEncoder.encode(registerDTO.getPassword()));
        user.setRealName(registerDTO.getRealName());
        user.setPhone(registerDTO.getPhone());
        user.setEmail(registerDTO.getEmail());
        user.setRole(2); // 默认为普通用户
        user.setStatus(1); // 默认启用
        
        save(user);
    }
    
    @Override
    public User login(LoginDTO loginDTO) {
        // 查询用户
        LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(User::getUsername, loginDTO.getUsername());
        User user = getOne(wrapper);
        
        if (user == null) {
            throw new RuntimeException("用户不存在");
        }
        
        // 验证密码
        if (!PasswordEncoder.matches(loginDTO.getPassword(), user.getPassword())) {
            throw new RuntimeException("密码错误");
        }
        
        // 检查用户状态
        if (user.getStatus() == 0) {
            throw new RuntimeException("账号已被禁用");
        }
        
        return user;
    }
    
    @Override
    public Page<User> getUserList(Integer page, Integer size, String username) {
        Page<User> pageParam = new Page<>(page, size);
        LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
        
        if (StringUtils.hasText(username)) {
            wrapper.like(User::getUsername, username);
        }
        
        wrapper.orderByDesc(User::getCreateTime);
        
        return page(pageParam, wrapper);
    }
}
```

**关键业务逻辑**：
1. **注册流程**：
   - 检查用户名是否已存在
   - 密码加密存储
   - 设置默认角色和状态

2. **登录流程**：
   - 根据用户名查询用户
   - 验证密码
   - 检查用户状态

3. **用户列表查询**：
   - 支持分页
   - 支持按用户名模糊搜索
   - 按创建时间倒序排列

### 4. JWT工具类（JwtUtil.java）

**文件位置**：`src/main/java/com/school/campusexpress/util/JwtUtil.java`

```java
package com.school.campusexpress.util;

import io.jsonwebtoken.Claims;
import io.jsonwebtoken.Jwts;
import io.jsonwebtoken.SignatureAlgorithm;
import io.jsonwebtoken.security.Keys;

import java.security.Key;
import java.util.Date;
import java.util.HashMap;
import java.util.Map;

public class JwtUtil {
    
    private static final Key SECRET_KEY = Keys.secretKeyFor(SignatureAlgorithm.HS256);
    private static final long EXPIRATION_TIME = 7 * 24 * 60 * 60 * 1000; // 7天
    
    /**
     * 生成JWT令牌
     */
    public static String generateToken(Long userId, String username, Integer role) {
        Map<String, Object> claims = new HashMap<>();
        claims.put("userId", userId);
        claims.put("username", username);
        claims.put("role", role);
        
        Date now = new Date();
        Date expiryDate = new Date(now.getTime() + EXPIRATION_TIME);
        
        return Jwts.builder()
                .setClaims(claims)
                .setIssuedAt(now)
                .setExpiration(expiryDate)
                .signWith(SECRET_KEY)
                .compact();
    }
    
    /**
     * 从令牌中获取用户ID
     */
    public static Long getUserIdFromToken(String token) {
        Claims claims = parseToken(token);
        return Long.valueOf(claims.get("userId").toString());
    }
    
    /**
     * 从令牌中获取用户名
     */
    public static String getUsernameFromToken(String token) {
        Claims claims = parseToken(token);
        return claims.get("username").toString();
    }
    
    /**
     * 从令牌中获取角色
     */
    public static Integer getRoleFromToken(String token) {
        Claims claims = parseToken(token);
        return Integer.valueOf(claims.get("role").toString());
    }
    
    /**
     * 验证令牌
     */
    public static boolean validateToken(String token) {
        try {
            parseToken(token);
            return true;
        } catch (Exception e) {
            return false;
        }
    }
    
    /**
     * 解析令牌
     */
    private static Claims parseToken(String token) {
        return Jwts.parserBuilder()
                .setSigningKey(SECRET_KEY)
                .build()
                .parseClaimsJws(token)
                .getBody();
    }
}
```

**JWT认证机制说明**：
1. **生成令牌**：登录成功后生成包含用户信息的JWT令牌
2. **验证令牌**：每次请求时验证令牌的有效性
3. **获取用户信息**：从令牌中解析出用户ID、用户名、角色等信息
4. **过期时间**：令牌默认7天过期

### 5. 密码加密工具（PasswordEncoder.java）

**文件位置**：`src/main/java/com/school/campusexpress/util/PasswordEncoder.java`

```java
package com.school.campusexpress.util;

import java.security.MessageDigest;
import java.security.NoSuchAlgorithmException;
import java.security.SecureRandom;
import java.util.Base64;

public class PasswordEncoder {
    
    private static final int SALT_LENGTH = 16;
    private static final int ITERATIONS = 10000;
    private static final int KEY_LENGTH = 256;
    
    /**
     * 加密密码
     */
    public static String encode(String rawPassword) {
        try {
            SecureRandom random = new SecureRandom();
            byte[] salt = new byte[SALT_LENGTH];
            random.nextBytes(salt);
            
            byte[] hash = hash(rawPassword, salt);
            
            return Base64.getEncoder().encodeToString(salt) + ":" + 
                   Base64.getEncoder().encodeToString(hash);
        } catch (Exception e) {
            throw new RuntimeException("密码加密失败", e);
        }
    }
    
    /**
     * 验证密码
     */
    public static boolean matches(String rawPassword, String encodedPassword) {
        try {
            String[] parts = encodedPassword.split(":");
            byte[] salt = Base64.getDecoder().decode(parts[0]);
            byte[] hash = Base64.getDecoder().decode(parts[1]);
            
            byte[] testHash = hash(rawPassword, salt);
            
            return MessageDigest.isEqual(hash, testHash);
        } catch (Exception e) {
            return false;
        }
    }
    
    /**
     * 使用PBKDF2算法进行哈希
     */
    private static byte[] hash(String password, byte[] salt) throws NoSuchAlgorithmException {
        MessageDigest digest = MessageDigest.getInstance("SHA-256");
        byte[] hash = password.getBytes();
        
        for (int i = 0; i < ITERATIONS; i++) {
            digest.reset();
            digest.update(hash);
            digest.update(salt);
            hash = digest.digest();
        }
        
        byte[] result = new byte[KEY_LENGTH / 8];
        System.arraycopy(hash, 0, result, 0, result.length);
        
        return result;
    }
}
```

**密码安全机制**：
1. **盐值（Salt）**：为每个密码生成随机盐值
2. **PBKDF2算法**：使用PBKDF2算法进行多次迭代哈希
3. **不可逆加密**：密码加密后无法解密
4. **安全比较**：使用`MessageDigest.isEqual()`防止时序攻击

import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state: () => ({
    token: localStorage.getItem('token') || '',
    userInfo: JSON.parse(localStorage.getItem('userInfo') || '{}')
  }),
  getters: {
    isLoggedIn: (state) => !!state.token,
    isAdmin: (state) => state.userInfo.role === 0,
    isCourier: (state) => state.userInfo.role === 1,
    isUser: (state) => state.userInfo.role === 2,
    username: (state) => state.userInfo.username || '',
    realName: (state) => state.userInfo.realName || ''
  },
  actions: {
    setToken(token: string) {
      this.token = token
      localStorage.setItem('token', token)
    },
    setUserInfo(userInfo: any) {
      this.userInfo = userInfo
      localStorage.setItem('userInfo', JSON.stringify(userInfo))
    },
    logout() {
      this.token = ''
      this.userInfo = {}
      localStorage.removeItem('token')
      localStorage.removeItem('userInfo')
    }
  }
})

## 🔄 核心流程说明

### 1. 用户注册流程

```
用户填写注册信息
    ↓
前端表单验证
    ↓
调用注册API（POST /api/user/register）
    ↓
后端检查用户名是否已存在
    ↓
密码加密（PBKDF2 + Salt）
    ↓
保存用户到数据库
    ↓
返回注册成功
```

**关键点**：
- 用户名唯一性检查
- 密码安全加密
- 默认角色设置为普通用户

---

### 2. 用户登录流程

```
用户输入用户名和密码
    ↓
前端表单验证
    ↓
调用登录API（POST /api/user/login）
    ↓
后端查询用户信息
    ↓
验证密码（使用加密后的密码比对）
    ↓
检查用户状态
    ↓
生成JWT令牌（包含用户ID、用户名、角色）
    ↓
返回令牌和用户信息
    ↓
前端保存令牌到localStorage
    ↓
跳转到首页
```

**关键点**：
- 密码使用加密后的值比对
- JWT令牌包含用户身份信息
- 令牌保存在localStorage中

---

### 3. JWT认证流程

```
前端发起请求
    ↓
从localStorage获取token
    ↓
在请求头中添加Authorization: Bearer {token}
    ↓
后端拦截器验证token
    ↓
解析token获取用户信息
    ↓
验证token是否过期
    ↓
允许访问或拒绝访问
```

**关键点**：
- 每次请求都需要携带token
- token在请求头中传递
- 后端验证token的有效性

### 权限控制实现

**前端路由守卫**：
```typescript
router.beforeEach((to, from, next) => {
  const userStore = useUserStore()
  
  if (to.path === '/login') {
    next()
    return
  }
  
  if (!userStore.isLoggedIn) {
    next('/login')
    return
  }
  
  // 检查角色权限
  if (to.meta.role && !to.meta.role.includes(userStore.userInfo.role)) {
    ElMessage.error('权限不足')
    next('/')
    return
  }
  
  next()
})
```

## 📝 学习要点总结

### 必须掌握的概念

1. **JWT认证**：无状态的认证机制，令牌包含用户信息
2. **密码加密**：使用PBKDF2算法，加盐哈希，不可逆
3. **角色权限**：基于角色的访问控制（RBAC）
4. **前后端分离**：通过RESTful API进行交互
5. **状态管理**：使用Pinia管理全局用户状态

### 答辩要点

1. **为什么使用JWT？**
   - 无状态，易于扩展
   - 跨域支持好
   - 不需要服务器存储会话

2. **密码如何保证安全？**
   - 使用PBKDF2算法
   - 加盐防止彩虹表攻击
   - 多次迭代增加破解难度

3. **如何实现权限控制？**
   - 前端：路由守卫 + 角色判断
   - 后端：注解 + 拦截器

## ❓ 常见问题

**Q1：JWT令牌过期了怎么办？**
A：前端检测到401错误时，跳转到登录页面重新登录。

**Q2：如何防止JWT令牌被盗用？**
A：使用HTTPS传输，设置合理的过期时间，可以考虑使用刷新令牌机制。

**Q3：密码忘记怎么办？**
A：当前版本不支持密码找回，可以联系管理员重置密码。

**Q4：如何实现记住我功能？**
A：延长JWT令牌的过期时间，或者使用长期有效的刷新令牌。
