## 🎯 学习目标

通过本章学习，你将能够：

## 📋 模块概述

数据统计模块负责对系统中的各类数据进行统计分析，包括：
- 快递数量统计（按状态、站点、时间等维度）
- 用户活跃度统计
- 站点运营数据统计
- 数据可视化展示

---

## 📊 数据库设计

### 统计数据来源

数据统计模块不单独创建表，而是从现有表中查询和计算统计数据：

1. **express表**：快递数量统计
2. **user表**：用户活跃度统计
3. **station表**：站点运营数据统计
4. **pickup_record表**：取件记录统计

### 1. 统计控制器（StatisticsController.java）

**文件位置**：`src/main/java/com/school/campusexpress/controller/StatisticsController.java`

```java
package com.school.campusexpress.controller;

import com.school.campusexpress.common.Result;
import com.school.campusexpress.dto.ExpressStatisticsDTO;
import com.school.campusexpress.dto.StationStatisticsDTO;
import com.school.campusexpress.dto.UserStatisticsDTO;
import com.school.campusexpress.service.StatisticsService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

import java.time.LocalDate;
import java.util.List;
import java.util.Map;

@RestController
@RequestMapping("/api/statistics")
@CrossOrigin
public class StatisticsController {
    
    @Autowired
    private StatisticsService statisticsService;
    
    /**
     * 获取快递统计概览
     */
    @GetMapping("/express/overview")
    public Result<?> getExpressOverview() {
        try {
            ExpressStatisticsDTO overview = statisticsService.getExpressOverview();
            return Result.success(overview);
        } catch (Exception e) {
            return Result.error(e.getMessage());
        }
    }
    
    /**
     * 获取快递趋势数据（按日期）
     */
    @GetMapping("/express/trend")
    public Result<?> getExpressTrend(
        @RequestParam(defaultValue = "7") Integer days
    ) {
        try {
            List<Map<String, Object>> trend = statisticsService.getExpressTrend(days);
            return Result.success(trend);
        } catch (Exception e) {
            return Result.error(e.getMessage());
        }
    }
    
    /**
     * 获取站点快递统计
     */
    @GetMapping("/station")
    public Result<?> getStationStatistics() {
        try {
            List<StationStatisticsDTO> statistics = statisticsService.getStationStatistics();
            return Result.success(statistics);
        } catch (Exception e) {
            return Result.error(e.getMessage());
        }
    }
    
    /**
     * 获取用户统计
     */
    @GetMapping("/user")
    public Result<?> getUserStatistics() {
        try {
            UserStatisticsDTO statistics = statisticsService.getUserStatistics();
            return Result.success(statistics);
        } catch (Exception e) {
            return Result.error(e.getMessage());
        }
    }
    
    /**
     * 获取取件趋势数据
     */
    @GetMapping("/pickup/trend")
    public Result<?> getPickupTrend(
        @RequestParam(defaultValue = "7") Integer days
    ) {
        try {
            List<Map<String, Object>> trend = statisticsService.getPickupTrend(days);
            return Result.success(trend);
        } catch (Exception e) {
            return Result.error(e.getMessage());
        }
    }
    
    /**
     * 获取综合统计数据
     */
    @GetMapping("/dashboard")
    public Result<?> getDashboardData() {
        try {
            Map<String, Object> data = statisticsService.getDashboardData();
            return Result.success(data);
        } catch (Exception e) {
            return Result.error(e.getMessage());
        }
    }
}
```

**关键接口说明**：
- `GET /api/statistics/express/overview`：获取快递统计概览
- `GET /api/statistics/express/trend`：获取快递趋势数据
- `GET /api/statistics/station`：获取站点快递统计
- `GET /api/statistics/user`：获取用户统计
- `GET /api/statistics/pickup/trend`：获取取件趋势数据
- `GET /api/statistics/dashboard`：获取综合统计数据

### 2. 统计服务实现（StatisticsServiceImpl.java）

**文件位置**：`src/main/java/com/school/campusexpress/service/impl/StatisticsServiceImpl.java`

```java
package com.school.campusexpress.service.impl;

import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;
import com.school.campusexpress.dto.ExpressStatisticsDTO;
import com.school.campusexpress.dto.StationStatisticsDTO;
import com.school.campusexpress.dto.UserStatisticsDTO;
import com.school.campusexpress.entity.Express;
import com.school.campusexpress.entity.PickupRecord;
import com.school.campusexpress.entity.Station;
import com.school.campusexpress.entity.User;
import com.school.campusexpress.mapper.ExpressMapper;
import com.school.campusexpress.mapper.PickupRecordMapper;
import com.school.campusexpress.mapper.StationMapper;
import com.school.campusexpress.mapper.UserMapper;
import com.school.campusexpress.service.StatisticsService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

import java.time.LocalDate;
import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;
import java.util.*;

@Service
public class StatisticsServiceImpl implements StatisticsService {
    
    @Autowired
    private ExpressMapper expressMapper;
    
    @Autowired
    private UserMapper userMapper;
    
    @Autowired
    private StationMapper stationMapper;
    
    @Autowired
    private PickupRecordMapper pickupRecordMapper;
    
    @Override
    public ExpressStatisticsDTO getExpressOverview() {
        ExpressStatisticsDTO dto = new ExpressStatisticsDTO();
        
        // 统计待取件数量
        LambdaQueryWrapper<Express> wrapper1 = new LambdaQueryWrapper<>();
        wrapper1.eq(Express::getStatus, 0);
        long pendingCount = expressMapper.selectCount(wrapper1);
        dto.setPendingCount(pendingCount);
        
        // 统计已取件数量
        LambdaQueryWrapper<Express> wrapper2 = new LambdaQueryWrapper<>();
        wrapper2.eq(Express::getStatus, 1);
        long pickedCount = expressMapper.selectCount(wrapper2);
        dto.setPickedCount(pickedCount);
        
        // 统计已取消数量
        LambdaQueryWrapper<Express> wrapper3 = new LambdaQueryWrapper<>();
        wrapper3.eq(Express::getStatus, 2);
        long cancelledCount = expressMapper.selectCount(wrapper3);
        dto.setCancelledCount(cancelledCount);
        
        // 统计总数量
        dto.setTotalCount(pendingCount + pickedCount + cancelledCount);
        
        // 统计今日新增
        LambdaQueryWrapper<Express> wrapper4 = new LambdaQueryWrapper<>();
        wrapper4.ge(Express::getCreateTime, LocalDate.now().atStartOfDay());
        long todayCount = expressMapper.selectCount(wrapper4);
        dto.setTodayCount(todayCount);
        
        return dto;
    }
    
    @Override
    public List<Map<String, Object>> getExpressTrend(Integer days) {
        List<Map<String, Object>> trend = new ArrayList<>();
        LocalDate today = LocalDate.now();
        DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd");
        
        for (int i = days - 1; i >= 0; i--) {
            LocalDate date = today.minusDays(i);
            LocalDateTime startOfDay = date.atStartOfDay();
            LocalDateTime endOfDay = date.plusDays(1).atStartOfDay();
            
            // 统计当日新增快递数量
            LambdaQueryWrapper<Express> wrapper = new LambdaQueryWrapper<>();
            wrapper.ge(Express::getCreateTime, startOfDay);
            wrapper.lt(Express::getCreateTime, endOfDay);
            long count = expressMapper.selectCount(wrapper);
            
            Map<String, Object> item = new HashMap<>();
            item.put("date", date.format(formatter));
            item.put("count", count);
            trend.add(item);
        }
        
        return trend;
    }
    
    @Override
    public List<StationStatisticsDTO> getStationStatistics() {
        List<StationStatisticsDTO> list = new ArrayList<>();
        
        // 查询所有启用的站点
        LambdaQueryWrapper<Station> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(Station::getStatus, 1);
        List<Station> stations = stationMapper.selectList(wrapper);
        
        for (Station station : stations) {
            StationStatisticsDTO dto = new StationStatisticsDTO();
            dto.setStationId(station.getId());
            dto.setStationName(station.getName());
            dto.setCapacity(station.getCapacity());
            
            // 统计待取件数量
            LambdaQueryWrapper<Express> wrapper1 = new LambdaQueryWrapper<>();
            wrapper1.eq(Express::getStationId, station.getId());
            wrapper1.eq(Express::getStatus, 0);
            long pendingCount = expressMapper.selectCount(wrapper1);
            dto.setPendingCount(pendingCount);
            
            // 统计已取件数量
            LambdaQueryWrapper<Express> wrapper2 = new LambdaQueryWrapper<>();
            wrapper2.eq(Express::getStationId, station.getId());
            wrapper2.eq(Express::getStatus, 1);
            long pickedCount = expressMapper.selectCount(wrapper2);
            dto.setPickedCount(pickedCount);
            
            // 统计已取消数量
            LambdaQueryWrapper<Express> wrapper3 = new LambdaQueryWrapper<>();
            wrapper3.eq(Express::getStationId, station.getId());
            wrapper3.eq(Express::getStatus, 2);
            long cancelledCount = expressMapper.selectCount(wrapper3);
            dto.setCancelledCount(cancelledCount);
            
            // 计算总数量
            long totalCount = pendingCount + pickedCount + cancelledCount;
            dto.setTotalCount(totalCount);
            
            // 计算使用率
            double usageRate = (double) totalCount / station.getCapacity() * 100;
            dto.setUsageRate(String.format("%.2f", usageRate));
            
            list.add(dto);
        }
        
        return list;
    }
    
    @Override
    public UserStatisticsDTO getUserStatistics() {
        UserStatisticsDTO dto = new UserStatisticsDTO();
        
        // 统计总用户数
        LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
        long totalCount = userMapper.selectCount(wrapper);
        dto.setTotalCount(totalCount);
        
        // 统计管理员数量
        LambdaQueryWrapper<User> wrapper1 = new LambdaQueryWrapper<>();
        wrapper1.eq(User::getRole, 0);
        long adminCount = userMapper.selectCount(wrapper1);
        dto.setAdminCount(adminCount);
        
        // 统计快递员数量
        LambdaQueryWrapper<User> wrapper2 = new LambdaQueryWrapper<>();
        wrapper2.eq(User::getRole, 1);
        long courierCount = userMapper.selectCount(wrapper2);
        dto.setCourierCount(courierCount);
        
        // 统计普通用户数量
        LambdaQueryWrapper<User> wrapper3 = new LambdaQueryWrapper<>();
        wrapper3.eq(User::getRole, 2);
        long userCount = userMapper.selectCount(wrapper3);
        dto.setUserCount(userCount);
        
        // 统计今日新增用户
        LambdaQueryWrapper<User> wrapper4 = new LambdaQueryWrapper<>();
        wrapper4.ge(User::getCreateTime, LocalDate.now().atStartOfDay());
        long todayCount = userMapper.selectCount(wrapper4);
        dto.setTodayCount(todayCount);
        
        return dto;
    }
    
    @Override
    public List<Map<String, Object>> getPickupTrend(Integer days) {
        List<Map<String, Object>> trend = new ArrayList<>();
        LocalDate today = LocalDate.now();
        DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd");
        
        for (int i = days - 1; i >= 0; i--) {
            LocalDate date = today.minusDays(i);
            LocalDateTime startOfDay = date.atStartOfDay();
            LocalDateTime endOfDay = date.plusDays(1).atStartOfDay();
            
            // 统计当日取件数量
            LambdaQueryWrapper<PickupRecord> wrapper = new LambdaQueryWrapper<>();
            wrapper.ge(PickupRecord::getPickupTime, startOfDay);
            wrapper.lt(PickupRecord::getPickupTime, endOfDay);
            wrapper.eq(PickupRecord::getStatus, 1);
            long count = pickupRecordMapper.selectCount(wrapper);
            
            Map<String, Object> item = new HashMap<>();
            item.put("date", date.format(formatter));
            item.put("count", count);
            trend.add(item);
        }
        
        return trend;
    }
    
    @Override
    public Map<String, Object> getDashboardData() {
        Map<String, Object> data = new HashMap<>();
        
        // 快递统计
        ExpressStatisticsDTO expressStats = getExpressOverview();
        data.put("expressStats", expressStats);
        
        // 用户统计
        UserStatisticsDTO userStats = getUserStatistics();
        data.put("userStats", userStats);
        
        // 站点统计
        List<StationStatisticsDTO> stationStats = getStationStatistics();
        data.put("stationStats", stationStats);
        
        // 快递趋势（最近7天）
        List<Map<String, Object>> expressTrend = getExpressTrend(7);
        data.put("expressTrend", expressTrend);
        
        // 取件趋势（最近7天）
        List<Map<String, Object>> pickupTrend = getPickupTrend(7);
        data.put("pickupTrend", pickupTrend);
        
        return data;
    }
}
```

**关键业务逻辑**：
1. **快递统计概览**：
   - 统计待取件、已取件、已取消的快递数量
   - 统计今日新增快递数量
   - 计算总数量

2. **快递趋势统计**：
   - 按日期统计快递数量
   - 支持自定义天数
   - 返回日期和数量列表

3. **站点统计**：
   - 统计每个站点的快递数量
   - 计算站点使用率
   - 按站点名称排序

4. **用户统计**：
   - 统计不同角色的用户数量
   - 统计今日新增用户
   - 计算用户分布

5. **取件趋势统计**：
   - 按日期统计取件数量
   - 支持自定义天数
   - 返回日期和数量列表

async function loadDashboardData() {
  loading.value = true
  try {
    const [expressRes, userRes, stationRes, expressTrendRes, pickupTrendRes] = await Promise.all([
      getExpressOverview(),
      getUserStatistics(),
      getStationStatistics(),
      getExpressTrend(7),
      getPickupTrend(7)
    ])
    
    Object.assign(expressStats, expressRes.data)
    Object.assign(userStats, userRes.data)
    stationStats.value = stationRes.data
    
    updateExpressChart(expressTrendRes.data)
    updatePickupChart(pickupTrendRes.data)
  } catch (error) {
    ElMessage.error('加载数据失败')
  } finally {
    loading.value = false
  }
}

function initCharts() {

## ❓ 常见问题

**Q1：统计数据不准确怎么办？**
A：检查数据库中的数据是否正确，检查统计逻辑是否有误。

**Q2：如何增加更多的统计维度？**
A：在统计服务中添加新的统计方法，在前端页面中添加对应的展示。

**Q3：图表不显示怎么办？**
A：检查ECharts是否正确初始化，检查数据格式是否正确，检查DOM元素是否存在。

**Q4：如何导出统计数据？**
A：可以在后端添加导出接口，使用Excel或PDF格式导出统计数据。
