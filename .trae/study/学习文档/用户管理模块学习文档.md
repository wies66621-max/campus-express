# 用户管理模块学习文档

## 1️⃣ 用户管理模块的整体架构

### 什么是用户管理模块？

用户管理模块是校园快递管理系统的管理员功能模块，允许管理员对系统用户进行管理，包括用户列表查询、用户详情查看、用户添加、修改、删除、状态管理、密码重置和搜索等功能。

### 用户管理模块的作用

| 功能 | 说明 |
|------|------|
| 用户列表查询 | 分页查询用户列表，支持角色和状态筛选 |
| 用户详情查询 | 查看指定用户的详细信息 |
| 用户添加 | 创建新用户账号 |
| 用户修改 | 更新用户信息 |
| 用户删除 | 删除用户账号（带安全限制） |
| 状态管理 | 启用/禁用用户账号 |
| 密码重置 | 重置用户密码为默认值（带安全限制） |
| 用户搜索 | 按用户名、手机号、真实姓名搜索用户 |

### 分层架构

```
┌─────────────────────────────────────┐
│         前端 (Vue3 + Element Plus)  │
└─────────────────────────────────────┘
                  ↓ HTTP请求
┌─────────────────────────────────────┐

│  - 返回响应                         │

                  ↓ 调用
┌─────────────────────────────────────┐
│       UserService (业务逻辑层)      │
│  - 业务逻辑处理                     │
│  - 安全检查                         │
│  - 数据校验                         │

└─────────────────────────────────────┘
                  ↓ 调用
┌─────────────────────────────────────┐
│        UserMapper (数据访问层)      │
│  - 数据库操作                       │
│  - SQL执行                          │
│  - 结果映射                         │
└─────────────────────────────────────┘
                  ↓ 操作

└─────────────────────────────────────┘
                  ↓ 存储
┌─────────────────────────────────────┐
│        MySQL 数据库                  │
└─────────────────────────────────────┘
```

### 为什么需要用户管理模块？

| 需求 | 说明 |
|------|------|
| 系统管理 | 管理员需要管理所有用户账号 |
| 权限控制 | 通过角色字段区分管理员和普通用户 |
| 安全管理 | 禁用违规用户，重置用户密码 |
| 数据维护 | 添加、修改、删除用户信息 |

```java
package com.school.campusexpress.controller;

import com.baomidou.mybatisplus.extension.plugins.pagination.Page;
import com.school.campusexpress.common.R;
import com.school.campusexpress.entity.User;
import com.school.campusexpress.service.UserService;
import io.swagger.v3.oas.annotations.Operation;
import io.swagger.v3.oas.annotations.Parameter;
import io.swagger.v3.oas.annotations.tags.Tag;
import jakarta.validation.Valid;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

@Tag(name = "管理员用户管理", description = "管理员用户管理接口")
@RestController
@RequestMapping("/admin/user")
public class AdminUserController {

    @Autowired
    private UserService userService;

    @Operation(summary = "用户列表查询")
    @GetMapping("/list")
    public R<Page<User>> listUsersByPage(
            @Parameter(description = "页码") @RequestParam(defaultValue = "1") Integer pageNum,
            @Parameter(description = "每页大小") @RequestParam(defaultValue = "10") Integer pageSize,
            @Parameter(description = "角色") String role,
            @Parameter(description = "状态") Integer status) {
        try {
            Page<User> page = userService.listUsersByPage(pageNum, pageSize, role, status);
            return R.success(page);
        } catch (RuntimeException e) {
            return R.error(e.getMessage());
        }
    }

    @Operation(summary = "用户详情查询")
    @GetMapping("/{id}")
    public R<User> getUserById(@Parameter(description = "用户ID") @PathVariable Long id) {
        try {
            User user = userService.getUserById(id);
            if (user != null) {
                user.setPassword(null);
                return R.success(user);
            }
            return R.error("用户不存在");
        } catch (RuntimeException e) {
            return R.error(e.getMessage());
        }
    }

    @Operation(summary = "添加用户")
    @PostMapping
    public R<String> addUser(@Valid @RequestBody User user) {
        try {
            boolean success = userService.addUser(user);
            if (success) {
                return R.success("添加成功");
            }
            return R.error("添加失败");
        } catch (RuntimeException e) {
            return R.error(e.getMessage());
        }
    }

    @Operation(summary = "更新用户信息")
    @PutMapping
    public R<String> updateUser(@Valid @RequestBody User user) {
        try {
            boolean success = userService.updateUser(user);
            if (success) {
                return R.success("更新成功");
            }
            return R.error("更新失败");
        } catch (RuntimeException e) {
            return R.error(e.getMessage());
        }
    }

    @Operation(summary = "删除用户")
    @DeleteMapping("/{id}")
    public R<String> deleteUser(
            @Parameter(description = "用户ID") @PathVariable Long id,
            @Parameter(description = "当前登录用户ID") @RequestParam Long currentUserId) {
        try {
            boolean success = userService.deleteUser(id, currentUserId);
            if (success) {
                return R.success("删除成功");
            }
            return R.error("删除失败");
        } catch (RuntimeException e) {
            return R.error(e.getMessage());
        }
    }

    @Operation(summary = "更新用户状态")
    @PutMapping("/status")
    public R<String> updateStatus(
            @Parameter(description = "用户ID") @RequestParam Long id,
            @Parameter(description = "状态(0=禁用,1=启用)") @RequestParam Integer status) {
        try {
            boolean success = userService.updateStatus(id, status);
            if (success) {
                return R.success("状态更新成功");
            }
            return R.error("状态更新失败");
        } catch (RuntimeException e) {
            return R.error(e.getMessage());
        }
    }

    @Operation(summary = "重置用户密码")
    @PutMapping("/reset-password")
    public R<String> resetPassword(@Parameter(description = "用户ID") @RequestParam Long id) {
        try {
            boolean success = userService.resetPassword(id);
            if (success) {
                return R.success("密码重置成功");
            }
            return R.error("密码重置失败");
        } catch (RuntimeException e) {
            return R.error(e.getMessage());
        }
    }

    @Operation(summary = "搜索用户")
    @GetMapping("/search")
    public R<Page<User>> searchUsers(
            @Parameter(description = "搜索关键词") @RequestParam String keyword,
            @Parameter(description = "页码") @RequestParam(defaultValue = "1") Integer pageNum,
            @Parameter(description = "每页大小") @RequestParam(defaultValue = "10") Integer pageSize) {
        try {
            Page<User> page = userService.searchUsers(pageNum, pageSize, keyword);
            return R.success(page);
        } catch (RuntimeException e) {
            return R.error(e.getMessage());
        }
    }
}
```

```java
package com.school.campusexpress.service.impl;

import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;
import com.baomidou.mybatisplus.extension.plugins.pagination.Page;
import com.baomidou.mybatisplus.extension.service.impl.ServiceImpl;
import com.school.campusexpress.entity.User;
import com.school.campusexpress.mapper.UserMapper;
import com.school.campusexpress.service.UserService;
import com.school.campusexpress.util.PasswordUtil;
import org.springframework.stereotype.Service;
import org.springframework.util.StringUtils;

import java.time.LocalDateTime;
import java.util.List;

@Service
public class UserServiceImpl extends ServiceImpl<UserMapper, User> implements UserService {

    @Override
    public Page<User> listUsersByPage(Integer pageNum, Integer pageSize, String role, Integer status) {
        Page<User> page = new Page<>(pageNum, pageSize);
        LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();

        if (StringUtils.hasText(role)) {
            wrapper.eq(User::getRole, role);
        }
        if (status != null) {
            wrapper.eq(User::getStatus, status);
        }

        wrapper.orderByDesc(User::getCreateTime);
        return page(page, wrapper);
    }

    @Override
    public boolean deleteUser(Long id, Long currentUserId) {
        if (id == null) {
            throw new RuntimeException("用户ID不能为空");
        }

        if (id.equals(currentUserId)) {
            throw new RuntimeException("不能删除当前登录用户");
        }

        User user = getById(id);
        if (user == null) {
            throw new RuntimeException("用户不存在");
        }

        return removeById(id);
    }

    @Override
    public boolean updateStatus(Long id, Integer status) {
        if (id == null) {
            throw new RuntimeException("用户ID不能为空");
        }
        if (status == null || (status != 0 && status != 1)) {
            throw new RuntimeException("状态值不正确");
        }

        User user = getById(id);
        if (user == null) {
            throw new RuntimeException("用户不存在");
        }

        user.setStatus(status);
        user.setUpdateTime(LocalDateTime.now());
        return updateById(user);
    }

    @Override
    public boolean resetPassword(Long id) {
        if (id == null) {
            throw new RuntimeException("用户ID不能为空");
        }

        User user = getById(id);
        if (user == null) {
            throw new RuntimeException("用户不存在");
        }

        if ("admin".equals(user.getRole())) {
            throw new RuntimeException("不能重置管理员密码");
        }

        String defaultPassword = "123456";
        user.setPassword(PasswordUtil.encode(defaultPassword));
        user.setUpdateTime(LocalDateTime.now());
        return updateById(user);
    }

    @Override
    public Page<User> searchUsers(Integer pageNum, Integer pageSize, String keyword) {
        if (keyword == null || keyword.trim().isEmpty()) {
            throw new RuntimeException("搜索关键词不能为空");
        }

        Page<User> page = new Page<>(pageNum, pageSize);
        LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
        wrapper.and(StringUtils.hasText(keyword), w -> w
                .like(User::getUsername, keyword)
                .or()
                .like(User::getPhone, keyword)
                .or()
                .like(User::getRealName, keyword)
        ).orderByDesc(User::getCreateTime);
        return page(page, wrapper);
    }
}
```

## 3️⃣ 用户管理模块的安全特性

### 安全特性1：防止管理员删除自己

#### 问题场景
管理员在删除用户时，可能会误删自己的账号，导致无法登录系统。

#### 解决方案
在删除用户时，检查要删除的用户ID是否与当前登录用户ID相同。

#### 代码实现

```java
@Override
public boolean deleteUser(Long id, Long currentUserId) {
    if (id == null) {
        throw new RuntimeException("用户ID不能为空");
    }

    if (id.equals(currentUserId)) {
        throw new RuntimeException("不能删除当前登录用户");
    }

    User user = getById(id);
    if (user == null) {
        throw new RuntimeException("用户不存在");
    }

    return removeById(id);
}
```

#### 答辩话术
"在用户删除功能中，我考虑到了一个安全问题：管理员可能会误删自己的账号。因此在Service层增加了一个安全检查，如果删除的用户ID与当前登录用户ID相同，就抛出异常，阻止删除操作。这样可以保证管理员账号的安全性。"

#### 测试用例

```http
DELETE http://localhost:8080/admin/user/2009442884314562562?currentUserId=2009442884314562562
```

响应：
```json
{
  "code": 500,
  "message": "不能删除当前登录用户",
  "data": null
}
```

### 安全特性2：防止重置管理员密码

#### 问题场景
管理员在重置用户密码时，可能会误重置其他管理员的密码，导致其他管理员无法登录。

#### 解决方案
在重置密码时，检查用户的角色是否为管理员，如果是管理员则拒绝重置。

#### 代码实现

```java
@Override
public boolean resetPassword(Long id) {
    if (id == null) {
        throw new RuntimeException("用户ID不能为空");
    }

    User user = getById(id);
    if (user == null) {
        throw new RuntimeException("用户不存在");
    }

    if ("admin".equals(user.getRole())) {
        throw new RuntimeException("不能重置管理员密码");
    }

    String defaultPassword = "123456";
    user.setPassword(PasswordUtil.encode(defaultPassword));
    user.setUpdateTime(LocalDateTime.now());
    return updateById(user);
}
```

#### 答辩话术
"在密码重置功能中，我考虑到了另一个安全问题：管理员可能会误重置其他管理员的密码。因此在Service层增加了一个安全检查，如果用户的角色是管理员，就抛出异常，阻止密码重置操作。这样可以保证管理员账号的安全性。"

#### 测试用例

```http
PUT http://localhost:8080/admin/user/reset-password?id=2009442884314562561
```

响应：
```json
{
  "code": 500,
  "message": "不能重置管理员密码",
  "data": null
}
```

### 答辩问题1：为什么要防止管理员删除自己？

**参考答案**：
"在实际应用中，管理员可能会误操作删除自己的账号，导致无法登录系统。为了防止这种情况，我在删除用户时增加了一个安全检查，如果删除的用户ID与当前登录用户ID相同，就抛出异常，阻止删除操作。这样可以保证管理员账号的安全性。"

### 答辩问题2：为什么要防止重置管理员密码？

**参考答案**：
"在实际应用中，管理员可能会误操作重置其他管理员的密码，导致其他管理员无法登录。为了防止这种情况，我在重置密码时增加了一个安全检查，如果用户的角色是管理员，就抛出异常，阻止密码重置操作。这样可以保证管理员账号的安全性。"

### 答辩问题3：搜索功能是如何实现的？

**参考答案**：
"搜索功能是在分页查询基础上的扩展，通过LambdaQueryWrapper构建多字段的模糊搜索条件，支持按用户名、手机号、真实姓名进行搜索。使用OR条件连接多个字段的LIKE查询，这样可以提高用户查找的效率。"

### 答辩问题4：为什么要隐藏密码字段？

**参考答案**：
"密码是敏感信息，不应该返回给前端。在Controller层，我将用户的密码字段设置为null后再返回。这样可以防止密码泄露，保证系统的安全性。"

### 答辩问题5：分页查询是如何实现的？

**参考答案**：
"分页查询使用MyBatis-Plus提供的Page类和LambdaQueryWrapper实现。Page类用于设置页码和每页大小，LambdaQueryWrapper用于构建查询条件。通过调用page(page, wrapper)方法，MyBatis-Plus会自动执行分页查询，返回包含分页信息的结果。"
