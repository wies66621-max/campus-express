# 校园快递管理系统 - 用户操作说明书

## 一、系统简介

### 1.1 系统概述
校园快递管理系统是一套为校园师生提供快递管理服务的应用系统，支持快递的录入、查询、取件等全流程管理。系统分为管理员端和用户端，分别提供不同的功能服务。

### 1.2 系统特点
- 操作简单，界面友好
- 功能完整，流程清晰
- 响应快速，稳定可靠
- 安全可靠，数据保护

## 二、系统运行环境

### 2.1 硬件要求
- CPU：双核及以上
- 内存：4GB 及以上
- 硬盘：20GB 及以上
- 网络：稳定的互联网连接

### 2.2 软件要求
- 操作系统：Windows 10/11、macOS 或 Linux
- 浏览器：Chrome、Firefox、Edge 或 Safari（推荐使用最新版本）
- 显示分辨率：1366×768 或更高

### 2.3 访问地址
- 本地访问：http://localhost:8080
- 局域网访问：http://服务器IP:8080

## 三、管理员操作指南

### 3.1 管理员登录

#### 3.1.1 登录步骤
1. 打开浏览器，输入系统访问地址
2. 点击"管理员登录"按钮
3. 输入管理员用户名和密码
4. 点击"登录"按钮
5. 登录成功后进入管理员首页

#### 3.1.2 登录注意事项
- 用户名和密码区分大小写
- 默认管理员账号：admin / admin123（首次登录后建议修改）
- 连续输错密码5次将锁定账户30分钟

### 3.2 快递录入

#### 3.2.1 功能说明
管理员可以录入新到的快递信息，系统会自动生成取件码。

#### 3.2.2 操作步骤
1. 登录管理员账号
2. 点击左侧菜单"快递管理"→"快递录入"
3. 填写快递信息：
   - **快递单号**：输入快递单号（必填，唯一）
   - **收件人姓名**：输入收件人姓名（必填）
   - **收件人电话**：输入收件人手机号（必填）
   - **快递站ID**：选择快递站（必填）
4. 点击"提交"按钮
5. 系统自动生成6位取件码
6. 记录取件码，告知收件人

#### 3.2.3 注意事项
- 快递单号不能重复，系统会自动校验
- 取件码生成后不可修改，请妥善保管
- 录入成功后系统会自动记录入库时间和操作管理员

### 3.3 快递查询

#### 3.3.1 功能说明
管理员可以查询所有快递信息，支持按条件筛选和分页显示。

#### 3.3.2 操作步骤
1. 登录管理员账号
2. 点击左侧菜单"快递管理"→"快递列表"
3. 选择查询条件（可选）：
   - **快递站**：选择要查询的快递站
   - **快递状态**：选择待取件、已取件或超期
4. 点击"查询"按钮
5. 查看查询结果，支持分页浏览

#### 3.3.3 查询结果说明
- 显示快递的详细信息：单号、收件人、电话、取件码、状态、入库时间等
- 状态说明：
  - **待取件**：快递已入库，等待取件
  - **已取件**：快递已被取走
  - **超期**：快递超过规定时间未取

### 3.4 快递详情查看

#### 3.4.1 功能说明
管理员可以查看快递的详细信息。

#### 3.4.2 操作步骤
1. 在快递列表中找到要查看的快递
2. 点击"查看详情"按钮
3. 查看快递的完整信息

### 3.5 快递信息修改

#### 3.5.1 功能说明
管理员可以修改快递的基本信息（快递单号和取件码不可修改）。

#### 3.5.2 操作步骤
1. 在快递列表中找到要修改的快递
2. 点击"编辑"按钮
3. 修改需要更新的信息
4. 点击"保存"按钮

#### 3.5.3 注意事项
- 快递单号和取件码不可修改
- 修改后系统会自动记录更新时间

### 3.6 快递删除

#### 3.6.1 功能说明
管理员可以删除错误的快递记录。

#### 3.6.2 操作步骤
1. 在快递列表中找到要删除的快递
2. 点击"删除"按钮
3. 确认删除操作

#### 3.6.3 注意事项
- 删除操作不可恢复，请谨慎操作
- 建议先确认快递信息是否正确再删除

### 3.7 管理员信息管理

#### 3.7.1 修改密码
1. 点击右上角用户名
2. 选择"修改密码"
3. 输入原密码和新密码
4. 点击"确认"按钮

#### 3.7.2 退出登录
1. 点击右上角用户名
2. 选择"退出登录"
3. 确认退出

## 四、用户操作指南

### 4.1 用户注册

#### 4.1.1 注册步骤
1. 打开浏览器，输入系统访问地址
2. 点击"用户注册"按钮
3. 填写注册信息：
   - **用户名**：输入用户名（必填，唯一）
   - **密码**：输入密码（必填，6-20位）
   - **确认密码**：再次输入密码（必填）
   - **手机号**：输入手机号（必填）
4. 点击"注册"按钮
5. 注册成功后自动登录

#### 4.1.2 注意事项
- 用户名不能重复
- 密码长度为6-20位
- 手机号用于快递查询，请填写真实号码

### 4.2 用户登录

#### 4.2.1 登录步骤
1. 打开浏览器，输入系统访问地址
2. 点击"用户登录"按钮
3. 输入用户名和密码
4. 点击"登录"按钮
5. 登录成功后进入用户首页

### 4.3 取件码取件

#### 4.3.1 功能说明
用户可以通过取件码取走自己的快递。

#### 4.3.2 操作步骤
1. 登录用户账号
2. 点击"取件服务"→"取件码取件"
3. 输入6位取件码
4. 点击"取件"按钮
5. 系统验证取件码
6. 取件成功后，快递状态更新为"已取件"

#### 4.3.3 注意事项
- 取件码为6位数字，请准确输入
- 取件码只能使用一次
- 取件成功后无法撤销

### 4.4 我的快递

#### 4.4.1 功能说明
用户可以查看自己名下的所有快递。

#### 4.4.2 操作步骤
1. 登录用户账号
2. 点击"我的快递"
3. 查看快递列表
4. 支持分页浏览

#### 4.4.3 快递状态说明
- **待取件**：快递已入库，等待取件
- **已取件**：快递已被取走
- **超期**：快递超过规定时间未取

### 4.5 按单号搜索快递

#### 4.5.1 功能说明
用户可以通过快递单号搜索快递信息。

#### 4.5.2 操作步骤
1. 登录用户账号
2. 点击"快递查询"→"按单号搜索"
3. 输入快递单号
4. 点击"搜索"按钮
5. 查看搜索结果

#### 4.5.3 注意事项
- 支持模糊搜索
- 可以输入部分单号进行搜索

### 4.6 按手机号搜索快递

#### 4.6.1 功能说明
用户可以通过手机号搜索快递信息。

#### 4.6.2 操作步骤
1. 登录用户账号
2. 点击"快递查询"→"按手机号搜索"
3. 输入手机号
4. 点击"搜索"按钮
5. 查看搜索结果

#### 4.6.3 注意事项
- 手机号需要完整输入
- 可以查询到该手机号对应的所有快递

### 4.7 用户信息管理

#### 4.7.1 修改密码
1. 点击右上角用户名
2. 选择"修改密码"
3. 输入原密码和新密码
4. 点击"确认"按钮

#### 4.7.2 退出登录
1. 点击右上角用户名
2. 选择"退出登录"
3. 确认退出

## 五、常见问题解答

### 5.1 登录问题

**Q1：忘记密码怎么办？**
A：请联系系统管理员重置密码。

**Q2：登录时提示"用户名或密码错误"？**
A：请检查用户名和密码是否正确，注意大小写。

**Q3：账户被锁定怎么办？**
A：连续输错密码5次会锁定账户30分钟，请等待后重试或联系管理员。

### 5.2 快递录入问题

**Q1：快递单号重复怎么办？**
A：系统会自动检测快递单号是否重复，如果重复会提示错误，请检查单号是否正确。

**Q2：取件码忘记了怎么办？**
A：可以在快递列表中查看快递详情，取件码会显示在详情页。

**Q3：录入错误怎么办？**
A：可以在快递列表中找到该快递，点击"编辑"按钮修改信息。

### 5.3 取件问题

**Q1：取件码输入错误怎么办？**
A：请检查取件码是否正确，取件码为6位数字。

**Q2：取件码失效了怎么办？**
A：取件码只能使用一次，如果已经取件成功则无法再次使用。如果未取件但提示失效，请联系管理员。

**Q3：可以代取快递吗？**
A：当前版本仅支持本人取件，代取功能将在后续版本中添加。

### 5.4 查询问题

**Q1：搜索不到快递怎么办？**
A：请检查搜索条件是否正确，可以尝试使用部分关键词进行模糊搜索。

**Q2：快递列表显示不全怎么办？**
A：系统支持分页显示，可以点击"下一页"查看更多快递。

### 5.5 系统问题

**Q1：系统无法访问怎么办？**
A：请检查网络连接是否正常，确认服务器是否正常运行。

**Q2：页面加载缓慢怎么办？**
A：可能是网络问题，请检查网络连接或稍后重试。

**Q3：遇到其他问题怎么办？**
A：请联系系统管理员或技术支持人员。

## 七、系统安全提示

### 7.1 账号安全
- 不要将账号密码告知他人

- 使用复杂密码（包含字母、数字、符号）

### 7.2 数据安全
- 不要在公共设备上保存密码
- 取件完成后及时退出登录
- 注意保护个人信息

### 7.3 操作安全
- 操作前确认信息是否正确
- 重要操作前进行二次确认
- 发现异常情况及时联系管理员
