```java
@Service
public class SmsService {
    
    @Value("${aliyun.sms.accessKeyId}")
    private String accessKeyId;
    
    @Value("${aliyun.sms.accessKeySecret}")
    private String accessKeySecret;
    
    @Value("${aliyun.sms.signName}")
    private String signName;
    
    @Value("${aliyun.sms.templateCode}")
    private String templateCode;
    
    public void sendSms(String phoneNumber, String code) {
        DefaultProfile profile = DefaultProfile.getProfile(
            "cn-hangzhou",
            accessKeyId,
            accessKeySecret
        );
        
        IAcsClient client = new DefaultAcsClient(profile);
        
        SendSmsRequest request = new SendSmsRequest();
        request.setPhoneNumbers(phoneNumber);
        request.setSignName(signName);
        request.setTemplateCode(templateCode);
        request.setTemplateParam("{\"code\":\"" + code + "\"}");
        
        try {
            SendSmsResponse response = client.getAcsResponse(request);
            if (!"OK".equals(response.getCode())) {
                throw new RuntimeException("短信发送失败：" + response.getMessage());
            }
        } catch (ServerException | ClientException e) {
            throw new RuntimeException("短信发送异常", e);
        }
    }
}
```

```java
@Service
public class ExcelImportService {
    
    public List<ExpressDTO> importExpress(MultipartFile file) throws IOException {
        List<ExpressDTO> list = new ArrayList<>();
        
        Workbook workbook = WorkbookFactory.create(file.getInputStream());
        Sheet sheet = workbook.getSheetAt(0);
        
        for (int i = 1; i <= sheet.getLastRowNum(); i++) {
            Row row = sheet.getRow(i);
            if (row == null) continue;
            
            ExpressDTO dto = new ExpressDTO();
            dto.setExpressNo(getCellValue(row, 0));
            dto.setRecipientName(getCellValue(row, 1));
            dto.setRecipientPhone(getCellValue(row, 2));
            dto.setStationId(Long.parseLong(getCellValue(row, 3)));
            
            list.add(dto);
        }
        
        workbook.close();
        return list;
    }
    
    private String getCellValue(Row row, int cellIndex) {
        Cell cell = row.getCell(cellIndex);
        if (cell == null) return "";
        
        switch (cell.getCellType()) {
            case STRING:
                return cell.getStringCellValue();
            case NUMERIC:
                return String.valueOf((long) cell.getNumericCellValue());
            default:
                return "";
        }
    }
}
```

```java
@Service
public class QrCodeService {
    
    public String generateQrCode(String content, int width, int height) throws Exception {
        Map<EncodeHintType, Object> hints = new HashMap<>();
        hints.put(EncodeHintType.CHARACTER_SET, "UTF-8");
        hints.put(EncodeHintType.MARGIN, 1);
        
        BitMatrix bitMatrix = new MultiFormatWriter().encode(
            content,
            BarcodeFormat.QR_CODE,
            width,
            height,
            hints
        );
        
        BufferedImage image = MatrixToImageWriter.toBufferedImage(bitMatrix);
        
        ByteArrayOutputStream baos = new ByteArrayOutputStream();
        ImageIO.write(image, "PNG", baos);
        
        return Base64.getEncoder().encodeToString(baos.toByteArray());
    }
}
```

```typescript
const BASE_URL = 'https://your-domain.com/api'

interface RequestOptions {
  url: string
  method?: 'GET' | 'POST' | 'PUT' | 'DELETE'
  data?: any
  header?: any
}

const request = (options: RequestOptions): Promise<any> => {
  const token = wx.getStorageSync('token')
  
  return new Promise((resolve, reject) => {
    wx.request({
      url: BASE_URL + options.url,
      method: options.method || 'GET',
      data: options.data,
      header: {
        'Authorization': `Bearer ${token}`,
        ...options.header
      },
      success: (res) => {
        if (res.statusCode === 200) {
          resolve(res.data)
        } else {
          reject(res.data)
        }
      },
      fail: (err) => {
        reject(err)
      }
    })
  })
}

export default request
```

#### 3.1 智能推荐

**需求描述**：
根据用户历史取件记录，推荐最佳取件时间。

**技术方案**：

1. **数据收集**
   - 记录用户取件时间
   - 统计取件时间段分布
   - 分析取件频率

2. **算法实现**

```java
@Service
public class RecommendationService {
    
    public String recommendPickupTime(Long userId) {
        // 获取用户历史取件记录
        LambdaQueryWrapper<PickupRecord> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(PickupRecord::getOperatorId, userId)
               .orderByDesc(PickupRecord::getPickupTime);
        List<PickupRecord> records = pickupRecordMapper.selectList(wrapper);
        
        if (records.isEmpty()) {
            return "暂无推荐";
        }
        
        // 统计取件时间段
        Map<Integer, Integer> timeSlotCount = new HashMap<>();
        for (PickupRecord record : records) {
            int hour = record.getPickupTime().getHour();
            timeSlotCount.put(hour, timeSlotCount.getOrDefault(hour, 0) + 1);
        }
        
        // 找出最常取件的时间段
        int bestHour = timeSlotCount.entrySet().stream()
            .max(Map.Entry.comparingByValue())
            .get()
            .getKey();
        
        return String.format("建议在%d:00-%d:00之间取件", bestHour, bestHour + 1);
    }
}
```

**技术方案**：

1. **数据分析维度**

```java
@Service
public class DataMiningService {
    
    public Map<String, Object> analyzeExpressData() {
        Map<String, Object> result = new HashMap<>();
        
        // 1. 日均快递量
        LambdaQueryWrapper<Express> wrapper1 = new LambdaQueryWrapper<>();
        wrapper1.ge(Express::getCreateTime, LocalDateTime.now().minusDays(30));
        long totalCount = expressMapper.selectCount(wrapper1);
        result.put("dailyAverage", totalCount / 30.0);
        
        // 2. 高峰时段分析
        Map<Integer, Long> hourlyCount = expressMapper.selectList(wrapper1).stream()
            .collect(Collectors.groupingBy(
                e -> e.getCreateTime().getHour(),
                Collectors.counting()
            ));
        int peakHour = hourlyCount.entrySet().stream()
            .max(Map.Entry.comparingByValue())
            .get()
            .getKey();
        result.put("peakHour", peakHour);
        
        // 3. 快递类型分布
        Map<String, Long> typeCount = expressMapper.selectList(wrapper1).stream()
            .collect(Collectors.groupingBy(
                e -> e.getExpressNo().substring(0, 2),
                Collectors.counting()
            ));
        result.put("typeDistribution", typeCount);
        
        return result;
    }
}
```

```java
@Service
public class ExpressServiceImpl implements ExpressService {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    @Override
    public Express getById(Long id) {
        String key = "express:" + id;
        
        // 先从缓存中获取
        Express express = (Express) redisTemplate.opsForValue().get(key);
        if (express != null) {
            return express;
        }
        
        // 缓存中没有，从数据库获取
        express = expressMapper.selectById(id);
        
        // 存入缓存，设置过期时间1小时
        redisTemplate.opsForValue().set(key, express, 1, TimeUnit.HOURS);
        
        return express;
    }
    
    @Override
    @Transactional
    public void updateById(Express express) {
        expressMapper.updateById(express);
        
        // 更新缓存
        String key = "express:" + express.getId();
        redisTemplate.opsForValue().set(key, express, 1, TimeUnit.HOURS);
    }
    
    @Override
    @Transactional
    public void removeById(Long id) {
        expressMapper.deleteById(id);
        
        // 删除缓存
        String key = "express:" + id;
        redisTemplate.delete(key);
    }
}
```

## 🔒 安全优化

### 1. 接口限流

**需求描述**：
防止接口被恶意调用，实现限流功能。

**技术方案**：

1. **使用Guava RateLimiter**

```java
@Component
public class RateLimiter {
    
    private final Map<String, com.google.common.util.concurrent.RateLimiter> limiters = new ConcurrentHashMap<>();
    
    public boolean tryAcquire(String key, int permitsPerSecond) {
        com.google.common.util.concurrent.RateLimiter limiter = limiters.computeIfAbsent(
            key,
            k -> com.google.common.util.concurrent.RateLimiter.create(permitsPerSecond)
        );
        return limiter.tryAcquire();
    }
}
```

2. **使用AOP实现限流**

```java
@Aspect
@Component
public class RateLimiterAspect {
    
    @Autowired
    private RateLimiter rateLimiter;
    
    @Around("@annotation(rateLimit)")
    public Object around(ProceedingJoinPoint joinPoint, RateLimit rateLimit) throws Throwable {
        String key = rateLimit.key();
        int permitsPerSecond = rateLimit.permitsPerSecond();
        
        if (!rateLimiter.tryAcquire(key, permitsPerSecond)) {
            throw new RuntimeException("请求过于频繁，请稍后再试");
        }
        
        return joinPoint.proceed();
    }
}
```

3. **使用注解**

```java
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface RateLimit {
    String key() default "";
    int permitsPerSecond() default 10;
}
```

### 2. 数据加密

1. **使用AES加密**

```java
@Component
public class EncryptionUtil {
    
    private static final String KEY = "YourSecretKey12345";
    private static final String IV = "YourIV1234567890";
    
    public String encrypt(String data) throws Exception {
        Cipher cipher = Cipher.getInstance("AES/CBC/PKCS5Padding");
        SecretKeySpec keySpec = new SecretKeySpec(KEY.getBytes(), "AES");
        IvParameterSpec ivSpec = new IvParameterSpec(IV.getBytes());
        cipher.init(Cipher.ENCRYPT_MODE, keySpec, ivSpec);
        byte[] encrypted = cipher.doFinal(data.getBytes());
        return Base64.getEncoder().encodeToString(encrypted);
    }
    
    public String decrypt(String encryptedData) throws Exception {
        Cipher cipher = Cipher.getInstance("AES/CBC/PKCS5Padding");
        SecretKeySpec keySpec = new SecretKeySpec(KEY.getBytes(), "AES");
        IvParameterSpec ivSpec = new IvParameterSpec(IV.getBytes());
        cipher.init(Cipher.DECRYPT_MODE, keySpec, ivSpec);
        byte[] decrypted = cipher.doFinal(Base64.getDecoder().decode(encryptedData));
        return new String(decrypted);
    }
}
```

### 2. 日志收集

#### 2.1 ELK Stack

**技术方案**：

1. **Logback配置**

```xml
<appender name="LOGSTASH" class="net.logstash.logback.appender.LogstashTcpSocketAppender">
    <destination>localhost:5000</destination>
    <encoder class="net.logstash.logback.encoder.LogstashEncoder"/>
</appender>

<root level="INFO">
    <appender-ref ref="LOGSTASH"/>
</root>
```

2. **Filebeat配置**

```yaml
filebeat.inputs:
- type: log
  enabled: true
  paths:
    - /var/log/campus-express/*.log
  fields:
    app: campus-express
  fields_under_root: true

output.logstash:
  hosts: ["localhost:5044"]
```
