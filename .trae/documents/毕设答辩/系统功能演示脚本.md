# 校园快递管理系统 - 系统功能演示脚本

## 演示准备

* 确保后端服务已启动（端口8081）

* 确保前端服务已启动（端口3001）

* 准备测试数据：

  * 管理员账号：admin / 123456

  * 普通用户账号：user1 / 123456

## 演示流程

### 一、管理员功能演示（5分钟）

* 打开浏览器，访问 <http://localhost:3001/>

* 输入管理员账号：admin

## 常见问题准备

### 技术类问题

1. 为什么选择Spring Boot + Vue技术栈？

   * Spring Boot：快速开发、配置简单、生态完善

   * Vue：轻量级、易学习、组件化开发

   * 前后端分离：解耦、可扩展、易维护

2. 如何保证系统安全性？

   * JWT Token认证

   * 密码加密存储

   * 接口权限控制

   * SQL注入防护

3. 如何处理并发取件？

   * 数据库事务控制

   * 取件码唯一性约束

   * 状态检查机制

### 设计类问题

1. 为什么普通用户首页不显示图表？

   * 普通用户关注个人快递信息，而非全站统计

   * 图表对用户决策意义不大

   * 简化界面，提升用户体验

2. 如何扩展系统功能？

   * 模块化设计，易于扩展

   * RESTful API，便于集成

   * 可增加短信通知、快递追踪等功能

### 实现类问题

1. 取件码如何生成？

   * 6位随机数字

   * 检查唯一性

   * 最多重试10次

2. 如何实现角色权限控制？

   * 基于角色的访问控制（RBAC）

   * JWT Token包含角色信息

   * 接口级别权限验证
