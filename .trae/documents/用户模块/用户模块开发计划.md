# 用户模块开发计划

## 项目结构分析
- Spring Boot 3.5.9 + Java 17
- MyBatis-Plus 3.5.5 (已配置)
- MySQL数据库 (campus_express)
- 统一返回结果类R.java已存在

## 开发步骤

### 1️⃣ 实体层（Entity / POJO）
- 在`src/main/java/com/school/campusexpress/entity`目录下创建`User.java`
- 使用`@TableName("user")`注解
- 包含字段：id、username、password、phone、role、status、create_time等
- 使用Lombok注解简化代码

### 2️⃣ Mapper层
- 在`src/main/java/com/school/campusexpress/mapper`目录下创建`UserMapper.java`
- 继承`BaseMapper<User>`
- 配置`@Mapper`注解

### 3️⃣ Service层
- 在`src/main/java/com/school/campusexpress/service`目录下：
  - 创建`UserService.java`接口
  - 创建`UserServiceImpl.java`实现类
- 实现功能：
  - 用户注册（含用户名重复校验）
  - 用户登录（账号 + 密码）
  - 根据id查询用户
  - 用户列表查询

### 4️⃣ Controller层
- 在`src/main/java/com/school/campusexpress/controller`目录下创建`UserController.java`
- 提供REST接口：
  - POST /user/register
  - POST /user/login
  - GET /user/{id}
  - GET /user/list
- 使用统一返回结果类R

### 5️⃣ 校验与安全
- 登录时进行空值校验
- 密码暂时明文存储
- 返回时不包含密码字段

### 6️⃣ 配置和启动
- 在主启动类添加`@MapperScan`注解
- 运行测试验证功能

## 预期交付物
1. User.java - 实体类
2. UserMapper.java - 数据访问层
3. UserService.java - 业务接口
4. UserServiceImpl.java - 业务实现
5. UserController.java - 控制器
6. 配置完成的应用程序
