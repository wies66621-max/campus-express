# 用户管理模块接口文档

## 模块概述
用户管理模块提供管理员对系统用户进行管理的功能，包括用户查询、添加、更新、删除、状态管理、密码重置等操作。

## 技术栈
- 后端框架：Spring Boot 3.1.5
- ORM框架：MyBatis-Plus 3.5.5
- 数据库：MySQL 8
- 接口文档：Swagger/Knife4j

## 接口列表

### 1. 用户列表查询
**接口地址**：`GET /admin/user/list`

**请求参数**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pageNum | Integer | 否 | 页码，默认1 |
| pageSize | Integer | 否 | 每页数量，默认10 |
| role | String | 否 | 角色筛选（admin/user） |
| status | Integer | 否 | 状态筛选（0-禁用，1-启用） |

**响应示例**：
```json
{
  "code": 200,
  "message": "成功",
  "data": {
    "records": [
      {
        "id": 2009444291478319105,
        "username": "newuser",
        "realName": "新用户",
        "phone": "13900139000",
        "role": "user",
        "status": 1,
        "createTime": "2026-01-09T09:57:07",
        "updateTime": "2026-01-09T09:57:07"
      }
    ],
    "total": 4,
    "size": 10,
    "current": 1,
    "pages": 1
  }
}
```

### 3. 添加用户
**接口地址**：`POST /admin/user/add`

**请求参数**：
```json
{
  "username": "testuser",
  "password": "123456",
  "realName": "测试用户",
  "phone": "13800138000",
  "role": "user"
}
```

**参数说明**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| username | String | 是 | 用户名，唯一 |
| password | String | 是 | 密码 |
| realName | String | 是 | 真实姓名 |
| phone | String | 是 | 手机号 |
| role | String | 是 | 角色（admin/user） |

**响应示例**：
```json
{
  "code": 200,
  "message": "成功",
  "data": "添加成功"
}
```

**安全特性**：密码自动加密存储

---

### 8. 用户搜索
**接口地址**：`GET /admin/user/search`

**请求参数**：
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pageNum | Integer | 否 | 页码，默认1 |
| pageSize | Integer | 否 | 每页数量，默认10 |
| keyword | String | 是 | 搜索关键字 |

**搜索范围**：
- 用户名（模糊匹配）
- 手机号（模糊匹配）
- 真实姓名（模糊匹配）

**响应示例**：
```json

  "data": {
    "records": [
      {
        "id": 2009593592443645954,
        "username": "testuser4",
        "realName": "测试用户4",

        "role": "user",
        "status": 1,
```

## 安全设计

### 1. 管理员不能删除自己
**实现位置**：[UserServiceImpl.java](file:///d:\IDEAProjects\campus-express\src\main\java\com\school\campusexpress\service\impl\UserServiceImpl.java#L98-L101)

**实现方式**：
```java
if (id.equals(currentUserId)) {
    throw new RuntimeException("不能删除当前登录用户");
}
```

**答辩话术**："系统设计了安全机制，管理员不能删除自己，防止误操作导致系统无法管理。"

---

### 2. 管理员不能重置其他管理员的密码
**实现位置**：[UserServiceImpl.java](file:///d:\IDEAProjects\campus-express\src\main\java\com\school\campusexpress\service\impl\UserServiceImpl.java#L122-L125)

**实现方式**：
```java
if ("admin".equals(user.getRole())) {
    throw new RuntimeException("不能重置管理员密码");
}
```

**答辩话术**："为了系统安全，管理员只能重置普通用户的密码，不能重置其他管理员的密码，防止权限滥用。"

---

### 3. 密码字段不返回
**实现位置**：[AdminUserController.java](file:///d:\IDEAProjects\campus-express\src\main\java\com\school\campusexpress\controller\AdminUserController.java#L30-L33)

**实现方式**：在返回用户详情前将密码字段置空

**答辩话术**："为了保护用户隐私，接口返回时自动隐藏密码字段。"

---

### 4. 密码加密存储
**实现位置**：[UserServiceImpl.java](file:///d:\IDEAProjects\campus-express\src\main\java\com\school\campusexpress\service\impl\UserServiceImpl.java#L75-L77)

**实现方式**：使用BCrypt算法对密码进行加密

**答辩话术**："系统使用BCrypt算法对用户密码进行加密存储，确保密码安全。"
